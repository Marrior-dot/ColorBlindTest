Teste de Ishihara

Sobre o Teste

O Teste de Ishihara é usado para verifcar se uma pessoa é ou não portadora de Daltonismo, o teste é feito a partir da observação de placas coloridas com números e símbolos que irão mostrar imagens diferentes dependendo da capacidade de enxergar cores de uma pessoa.

Descrição

Os princípos utilizados para a realização dos testes a seguir foram baseados nos estudos das seguintes Referências.Portanto, é recomendado que se consulte um Oftalmologista para resultados precisos e oficiais.

Instruções

aperte no botão Começar Teste observe as 7 imagens que aparecerão,mantenha a tela a uma distância de 50cm e à altura dos olhos, após isso, veja os resultados baseado no que você observou.

Resultados

Primeira Imagem
  • Visão Normal:enxerga um 74
  • Daltonsimo:enxerga um 21
  • Possível diagnóstico:visão com deficiência para enxergar o verde e o vermelho(Deuteranopia ou Protanomalia)

Segunda Imagem
  • Visão Normal:enxerga um 42
  • Daltonismo:dificuldade em enxergar o 4 ou o 2
  • Possível diagnóstico:Protanomalia(daltonismo leve para vermeho) ou Deuteranopia(daltonismo de verde) para quem enxerga o 4. Protanopia(daltonismo de vermelho) ou Deuteranomalia(daltonismo leve de verde) para quem enxerga o 2

Terceira Imagem
  • Visão Normal:enxerga um 6
  • Daltonismo: não enxerga nenhum número
  • Possível diagnóstico:daltonismo tanto para verde(Deuteranopia) quanto para vermelho (Protanomalia)

Quarta Imagem
  • Visão Normal:Não enxerga nenhum número
  • Daltonismo:enxerga um 45
  • Possível diagnóstico:Daltonismo para verde(Deuteranopia) ou vermelho(Protanomalia)

Quinta Imagem
  • Visão Normal:Enxerga um 29
  • Daltonismo:Enxergar um 70
  • Possível diagnóstico:Daltonismo para verde(Deuteranopia) e vermelho(Protanomalia)

Sexta Imagem
  • Visão Normal:Enxerga uma linha vermelha e outra roxa
  • Daltonismo:enxergará somente uma das linhas
  • Possível diagnóstico:Caso enxergue somente a linha vermelha Deuteranopia(Daltonismo para verde), caso enxergue somente a linha roxa,Protanomalia(Daltonismo para o vermelho)

Sétima Imagem
  • Visão Normal:Enxerga um 26
  • Daltonismo:Enxergará somente o 2 ou 6
  • Possível diagnóstico:Caso enxergue somente o 2,Deuteranopia(daltonismo para o verde), caso enxergue somente o 6 Protanomalia
